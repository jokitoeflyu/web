Item description

MaterialM Free Vuetify Nuxtjs Admin Template is an open-source dashboard built with Vuetify and Nuxt.js. It offers a clean, organized code making it easy for developers to customize and get started quickly. With responsive design and modern UI components, this admin panel template is perfect for building lightweight applications.

It also includes multiple pages, a range of useful UI elements, and helps you to start on a project without any delay or extra effort. Whether you're creating a simple app or starting on a large project, this Vuetify template and Nuxt template provide a strong base for building your project, and the best part is, they’re completely free to use.
